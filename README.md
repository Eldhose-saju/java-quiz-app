JAVA QUIZ APP



The Java Quiz Application is a desktop-based multiple-choice quiz system developed using Java Swing. It provides an interactive graphical interface where users can attempt quizzes, use a lifeline for assistance, and view their final score.
The application also includes admin-level controls for managing quiz content and users, along with a leaderboard system that displays top scores, encouraging competition and user engagement.



Features



User Features



Multiple-choice questions (MCQs)
Interactive GUI using Java Swing
Lifeline feature to assist users during the quiz
Navigation between questions
Automatic score calculation
Final result display


Admin Features


Add, edit, and delete quiz questions
Manage answer options and correct answers
Control user profiles and access
Maintain quiz content and integrity


Additional Features


Leaderboard to display top-scoring users
Score comparison for competitive learning


Technologies Used


Java
Java Swing – GUI development
AWT – Event handling
Object-Oriented Programming (OOP)
